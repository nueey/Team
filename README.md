#### 역할 및 업무 내용
+ ### 제작총괄(PM) : 박성호
 + 프로젝트 수행프로젝트 수행계획을 수립하고, 프로젝트 진행을 통제하여 목표한 제품/서비스를 제공을 책임짐.

+ 개발
 + 문예은 : 회원가입, 회원등급에 따른 로그인 페이지 작성
 + 유영대 : PT신청 등록 페이지 작성
 + 이수영 : 관리자 대시보드의 강사등록, 수정, 삭제가 가능한 강사 관리 페이지 작성
 + 김형진 : 관리자 대시보드의 식단 관리 페이지 작성
 + 박성호 : 관리자 대시보드의 스케줄 관리와 신청관리 페이지 및 메인페이지
